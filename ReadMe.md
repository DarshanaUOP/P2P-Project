## Start Server
Boostrap server runs on port 5555
```
 python3 Server.py 

```

## Start Client
```
python3 Client.py <ServerIp> <serverPort:5555>  <ClientIp> <ClientPort> <NameFortheClient> <sharedDirectory>
```

## Command

### Search Files
```
Command: search <some file name or part>
```
### List Files
```
Command: ls <path>
```
### Download Files
```
Command: fetch <user name> file path
```
### Neibour Table
```
Command: tb
```
### Change Neibour
```
Command: reset
```
### Help
```
Command: help
```
